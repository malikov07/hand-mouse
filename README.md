# 🇬🇧 English

## ✨ Features

- 🖱️ Move the mouse cursor with your palm
- 👆 Left Click & Drag using **Thumb + Index Finger**
- 👉 Right Click using **Thumb + Middle Finger**
- ✌️ Scroll pages using **Index + Middle Finger**
- 📷 Real-time webcam hand tracking
- ⚡ Powered by **MediaPipe Hand Landmarker**

---

## 📋 Requirements

- Python **3.9+**
- Webcam
- Windows / Linux / macOS

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/malikov07/hand-mouse.git
cd hand-mouse
```

### 2. Install dependencies

```bash
pip install opencv-python mediapipe pyautogui numpy
```
---

## ▶️ Run

```bash
python v.py
```

A webcam window will open.

Press **Q** to exit.

---

## 🎮 Controls

| Gesture | Action |
|----------|---------|
| ✋ Move your palm | Move cursor |
| 🤏 Thumb + Index Finger | Left Click / Drag |
| 🤌 Thumb + Middle Finger | Right Click |
| ✌️ Index + Middle Finger | Scroll Up / Down |

---

## ⚠️ Safety

If the cursor starts behaving unexpectedly:

> Simply move your **physical mouse** to **any corner of the screen**.

PyAutoGUI's fail-safe will immediately stop the program.
